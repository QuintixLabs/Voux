# Project Structure

- `src/` — backend code (API routes, embed, DB, middleware, utils etc).
- `public/` — static HTML, CSS, and JS for the public pages, dashboard, profile, and settings.
- `scripts/` — maintenance scripts.
- `data/` — SQLite database and config.
- `docs/` — contributor docs.
- `.github/` — issue templates and workflows.
- `Dockerfile*` — container builds.
