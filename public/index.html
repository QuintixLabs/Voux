Styling your counter

Styling your counter with Voux is super simple. All you need to do is wrap your counter script inside an element. We'll use a <span> in this example:

<span class="counter-widget">
  <!---------------------replace this with urs--------------------->
  <script async src="https://your-domain/embed/abc123.js"></script>
</span>

Once that's in place, you can style it however you like using CSS. Here's a simple example that centers the counter on the screen and makes the text black:

.counter-widget {
  position: fixed;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
  text-align: center;
  color: black;
  font-weight: 600;
  font-size: 3rem;
  font-family: system-ui, sans-serif;
}

And that's it. Your counter is now styled and ready to use. You can change the font, colors, or layout any way you like.

Self-hosting Voux

Running your own copy is easy. Use Node.js 22 (install and switch with fnm or any manager) and keep the files anywhere. The SQLite file sits in the project folder, so you do not need any other services.

1. Grab the project

git clone https://github.com/QuintixLabs/Voux.git
cd voux
npm install

2. Drop in your settings

cp .env.example .env
$EDITOR .env   # set ADMIN_TOKEN, etc.

3. Run it

npm run dev   # dev server with reload
npm start     # production build

4. Tweak optional vars

PORT=8787
PUBLIC_BASE_URL=https://counter.yourdomain.com

5. Share it with a Cloudflared tunnel, reverse proxy, or any static IP. Point your embed script at that URL and you are done.
