Voux · About

What is Voux?

Voux is a self-hosted counter generator. It runs on your hardware, lets you style embeds any way you want, and keeps the data with you.

Why I built it

There were no good free hit counters left, so I decided to make my own. A long time ago there was a site called Cutercounter where you could make simple unique-visit counters, but it shut down. So I switched to freehitcounters.org for a while, but I wanted something I could control myself. If a service goes down or changes, I don't want to depend on it. So I built Voux to be my own version that anyone can run and trust.

Self-hosted or community instances

You can run Voux yourself, or you can use a public instance hosted by someone else. Those hosts control their own databases, policies, and uptime. If you join their instance, you trust them, not us, with your data. We provide an official public instance list that meets our requirements, which you can find here.

Need help?

The code lives on GitHub. Open an issue, send a pull request, or fork it for your own needs. Everything is public and free.
