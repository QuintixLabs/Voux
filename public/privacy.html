Voux · Privacy

Privacy Policy

Voux runs on your hardware, so you control what data it stores. Here's what gets saved by default and how to manage it.

Last updated: 5 February 2026

What gets saved

The SQLite file keeps counter IDs, labels, values, and the visitor IP address with a last visit timestamp. IPs are stored in both counting modes. Nothing leaves your machine.

Unique visit mode

If you pick "Unique visitors" Voux stores visitor IPs and uses them to make sure each IP is only counted once. If the same IP visits again, the counter does not go up.

Every-visit mode

If you pick "Every-visit" Voux always stores visitor IPs. With a 0s throttle, every page load is counted, even from the same IP. If a throttle is set (for example 10s), the same IP can only increase the counter once per throttle window.

Cookies and tracking

The embed script doesn't set cookies or load third-party trackers. If you add any extras on your own site, explain them in your own policy.

You control retention

You decide how long to keep the database, who can access it, and what legal notices you show. If you let others embed counters, make sure your policy covers how their data is handled.
