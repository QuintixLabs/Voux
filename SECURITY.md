# Security Policy

## Supported Versions

We support the latest released version of **Voux**. Please update before reporting issues.

## Reporting a Vulnerability

For critical issues, please use the repository's [Security tab](https://github.com/QuintixLabs/Voux/security) to submit a private report to the maintainers.

For non-critical issues, open a public issue and mark it as security so the community can help. Please include:

- The version/commit where you found it
- Clear steps to reproduce
- Impact description and, if possible, suggested fixes

We're a small team and may reach out for more details. Thanks for helping keep **Voux** secure.
